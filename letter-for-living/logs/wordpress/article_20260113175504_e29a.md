# 프린터 인쇄 안될 때 점검 순서: 집·사무실에서 바로 따라 하는 체크리스트

프린터가 갑자기 인쇄되지 않으면 문서 작업이 멈추고 원인 파악에 시간이 많이 걸릴 수 있습니다. 이럴 때는 무작정 재설치하기보다, **프린터 인쇄 안될 때 점검 순서**를 정해 단계별로 확인하는 것이 효율적입니다. 아래 순서는 Windows 기준으로도, 공용(네트워크) 프린터 환경에서도 대부분 공통적으로 적용됩니다.

## 프린터 인쇄 안될 때 점검 순서가 필요한 이유(원인/배경)

프린터 인쇄 문제는 크게 **연결 문제(전원·케이블·Wi‑Fi)**, **대기열/드라이버 문제**, **소모품/용지 문제**, **네트워크 공유 설정 문제**로 나뉘는 경우가 많습니다. 특히 “인쇄 버튼은 눌렀는데 반응이 없다”는 상황은 대기열이 멈췄거나 기본 프린터가 바뀌었을 때 자주 발생합니다. 반면 “용지가 걸렸다/토너가 부족하다”처럼 기기 자체 경고가 있는 경우는 하드웨어 쪽 점검이 우선입니다.

## 프린터 인쇄 안될 때 점검 순서(단계별 해결)

### 1) 가장 먼저: 전원·상태 표시·오류 메시지 확인
프린터 전원이 켜져 있는지, 전면 패널에 **오류 코드/경고등(용지 걸림, 커버 열림, 토너/잉크 부족)**이 표시되는지 확인합니다. 경고가 있으면 해당 문제를 먼저 해결해야 인쇄가 정상 진행됩니다. 가능하면 프린터 화면에서 “준비” 상태인지 확인해 주세요.

### 2) 연결 확인: USB/네트워크/Wi‑Fi 점검
USB 프린터라면 케이블이 느슨하지 않은지 확인하고, 다른 USB 포트로 바꿔 연결해 보시는 것이 좋습니다. 네트워크 프린터라면 프린터의 IP가 바뀌었거나 Wi‑Fi가 끊긴 경우가 있어, 같은 공유기에 정상 연결되어 있는지 확인합니다. 사무실 공용 프린터는 네트워크 장애나 프린터 서버 문제도 원인이 될 수 있습니다.

### 3) 기본 프린터 설정과 오프라인 상태 해제
Windows에서 **설정 > Bluetooth 및 장치 > 프린터 및 스캐너**로 들어가 사용 중인 프린터가 선택되어 있는지 확인합니다. “오프라인”으로 표시되면 프린터 전원/네트워크 상태를 다시 확인한 뒤, 가능한 경우 오프라인 해제 또는 다시 연결을 진행합니다. 프린터가 여러 대 등록돼 있으면 기본 프린터가 다른 장치로 바뀌어 인쇄가 안 되는 것처럼 보일 수 있습니다.

### 4) 인쇄 대기열(스풀러) 확인 및 문서 취소
**프린터 인쇄 안될 때 점검 순서**에서 가장 효과가 큰 단계 중 하나가 대기열 확인입니다. 인쇄 작업이 “오류”, “일시 중지”, “삭제 중” 상태로 걸려 있으면 다음 작업이 진행되지 않습니다.  
- Windows에서 해당 프린터를 선택한 뒤 **인쇄 대기열 열기**로 들어가 걸린 문서를 취소해 보세요.  
- 대기열이 계속 멈추면 PC 재부팅 또는 프린터 전원 껐다 켜기를 함께 진행해 보시는 것이 좋습니다.

### 5) 테스트 페이지 출력으로 문제 범위 좁히기
문서 프로그램(한글/워드/브라우저)에서만 안 되는지 확인하려면 **테스트 페이지**를 출력해 보세요. 테스트 페이지가 인쇄되면 프린터 자체보다는 프로그램/문서 설정(용지 크기, 인쇄 대상 프린터 선택 등) 문제일 가능성이 큽니다. 테스트 페이지도 안 나오면 연결, 드라이버, 대기열, 하드웨어 쪽 점검을 이어가야 합니다.

### 6) 드라이버/포트 문제 점검(필요 시 재설치)
드라이버가 손상되었거나 포트가 바뀌면 인쇄가 멈출 수 있습니다. 장치 관리자 또는 프린터 속성에서 드라이버 상태를 확인하고, 제조사에서 제공하는 최신 드라이버로 업데이트하는 방법이 일반적입니다. 네트워크 프린터는 포트가 “자동 검색”으로 바뀌면서 IP가 맞지 않는 경우도 있어 포트 설정을 확인해 보셔야 합니다.

### 7) 용지·토너/잉크·걸림 등 하드웨어 점검
용지함에 용지가 제대로 들어 있는지, 용지 가이드가 너무 조여 있지 않은지 확인합니다. 용지 걸림이 있었다면 걸린 종이를 제거한 뒤 내부에 찢어진 조각이 남지 않았는지도 점검해 주세요. 레이저 프린터는 토너가 부족해도 인쇄가 흐리게 나오거나 중간에 멈출 수 있어 소모품 상태 확인이 필요합니다.

### 8) 네트워크 공유 프린터라면: 다른 PC에서도 같은지 확인
사무실 공유 프린터는 특정 PC만 안 되는지, 모든 PC에서 안 되는지에 따라 원인이 달라집니다. 모든 PC에서 안 되면 프린터/네트워크 장비(공유기, 스위치) 문제일 수 있고, 한 대만 안 되면 해당 PC의 드라이버/대기열/권한 문제일 가능성이 큽니다. 가능하면 다른 PC에서 테스트 인쇄를 해 문제 범위를 좁혀 보세요.

## 자주 하는 실수 / FAQ

## Q1. 프린터를 재설치했는데도 인쇄가 안 됩니다. 다음 점검 순서는 무엇인가요?
재설치 후에도 동일하면 **대기열(스풀러) 정지**, **포트/IP 불일치**, **오프라인 상태**, **네트워크 연결**을 우선 확인해 보시는 것이 좋습니다. 특히 네트워크 프린터는 IP가 변경되면 재설치를 해도 포트가 맞지 않아 인쇄가 실패할 수 있습니다. 이 경우 프린터 속성의 포트 설정을 확인하는 것이 도움이 됩니다.

## Q2. “인쇄 중”으로 뜨는데 프린터에서 아무 반응이 없습니다.
대기열이 멈춰 있거나, 기본 프린터가 다른 장치로 지정되어 있는 경우가 많습니다. **프린터 인쇄 안될 때 점검 순서**대로 기본 프린터 확인 → 대기열 취소 → 테스트 페이지 출력 순으로 확인해 보세요. USB 연결이면 케이블/포트 변경도 함께 점검하는 편이 좋습니다.

## Q3. 특정 문서만 인쇄가 안 됩니다.
문서 자체의 용지 크기(A4/Letter), 여백, 글꼴 문제 또는 PDF 뷰어/브라우저 인쇄 설정 문제일 수 있습니다. 같은 프린터로 다른 문서가 정상 출력되면 프린터보다는 문서/프로그램 설정을 의심해 볼 만합니다. 먼저 “다른 파일 인쇄”와 “테스트 페이지”로 비교해 보세요.

## 요약

프린터가 인쇄되지 않을 때는 **프린터 인쇄 안될 때 점검 순서**를 정해 전원/오류 표시 → 연결 → 기본 프린터/오프라인 → 대기열 → 테스트 페이지 → 드라이버/포트 → 소모품·용지 순으로 확인하는 것이 효율적입니다. 단계별로 문제 범위를 좁히면 불필요한 재설치나 시간 낭비를 줄일 수 있습니다. 공유 프린터 환경이라면 다른 PC에서도 동일한지 확인해 원인을 빠르게 분리해 보시는 것이 좋습니다.
